静态方法使用泛形有时会报错：
```
public static Flowable<T> just(T item)
```
修改为这样就好了：
```
public static <T> Flowable<T> just(T item)
```
参考：https://stackoverflow.com/questions/4209080/using-generic-types-in-a-static-context
